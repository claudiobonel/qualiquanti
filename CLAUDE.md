# Chatbot de Análise de Dados

Sistema de chatbot interativo para análise exploratória de dados.
O usuário faz upload de arquivos xlsx/csv e conversa com o assistente sobre os dados.

## Comportamento

Usa a skill `analista-dados` para todas as interações com datasets carregados.
Responde sempre em português brasileiro.
Mantém contexto da conversa e dos dados ao longo de toda a sessão.
